Conectar CSS a tu HTML

Lo primero de lo primero · 10 minutos

🤔 ¿Cómo "habla" el HTML con el CSS?

Hay tres maneras de meter CSS en tu HTML:

1. Inline (en el atributo style): rápido pero feo, no se reutiliza.

2. Interno (dentro de <style>): útil para pruebas, todo en un archivo.

3. Externo (recomendado): creás un archivo .css y lo conectás con <link>.

<!DOCTYPE html>
<html>
<head>
  <!-- Forma 3: archivo externo (LO QUE SE USA) -->
  <link rel="stylesheet" href="estilos.css">

  <!-- Forma 2: interno -->
  <style>
    h1 { color: red; }
  </style>
</head>
<body>

  <!-- Forma 1: inline (NO se recomienda) -->
  <h1 style="color: red;">Hola</h1>

</body>
</html>

📝 La estructura de una regla CSS

Toda regla de CSS tiene 3 partes:

h1 {                       /* ← selector: a quién le hablo */
  color: blue;            /* ← propiedad: valor; */
  font-size: 32px;
  text-align: center;
}                          /* ← cierre con } */
⚠️ Cosas que se olvidan mucho:
  • Cada propiedad termina con punto y coma ;
  • Las llaves { } son obligatorias.
  • Los textos sin comillas (excepto cuando son font-family con espacios).

Selectores: a quién le estás hablando

El concepto más importante de CSS · 30 minutos

🎯 Las 4 formas más usadas de seleccionar

  • Por etiqueta: p { ... } → todos los <p> de la página.
  • Por clase: .destacado { ... } → todos los que tengan class="destacado".
  • Por id: #unico { ... } → el ÚNICO elemento con id="unico".
  • Combinados: article p { ... } → los <p> que están dentro de un <article>.
/* Por ETIQUETA: afecta a TODOS los párrafos */
p {
  color: gray;
  font-size: 16px;
}

/* Por CLASE: solo a los que tengan class="destacado" */
.destacado {
  background: yellow;
  font-weight: bold;
}

/* Por ID: solo al elemento con id="unico" */
#unico {
  color: red;
  text-transform: uppercase;
}

/* COMBINADO: párrafos DENTRO de article */
article p {
  font-style: italic;
}

/* Varios a la vez (separados por coma) */
h1, h2, h3 {
  color: navy;
}

Soy un h3 (afectado por etiqueta)

Soy un párrafo común, color gris.

Otro párrafo igual.

Yo tengo class="destacado" → fondo amarillo.

Yo tengo id="unico" → en mayúsculas.

🆔 ¿Cuándo usar clase y cuándo id?

  • Clase (.): para estilos que se REPITEN. Ejemplo: .boton, .tarjeta, .titulo-grande.
  • ID (#): para algo ÚNICO en la página. Ejemplo: #header-principal, #carrito.

En el día a día se usan muchísimo más las clases que los ids.

Colores y fondos

El alma visual de tu sitio · 30 minutos

🎨 Las 4 formas de escribir un color

  • Nombre: red, blue, tomato... (hay 140 nombres).
  • HEX: #ff5733 → la más común en proyectos reales.
  • RGB: rgb(255, 87, 51) → cada número va de 0 a 255.
  • RGBA: rgba(255, 87, 51, 0.5) → igual pero con transparencia (0 a 1).
.titulo     { color: red; }
.subtitulo  { color: #ff5733; }
.parrafo    { color: rgb(100, 100, 100); }
.transparente { color: rgba(0, 0, 0, 0.5); }

/* Color de fondo */
body {
  background-color: #f5ede0;
}

/* Imagen de fondo */
.hero {
  background-image: url("foto.jpg");
  background-size: cover;       /* que cubra todo */
  background-position: center;  /* centrada */
}

/* DEGRADADO (gradiente) - muy de moda */
.gradiente {
  background: linear-gradient(45deg, #ff5733, #1572b6);
}

Texto y tipografía

Cómo se ve cada letra · 30 minutos

🔤 Las propiedades de texto que más vas a usar

  • font-size → tamaño de letra (16px, 1.2rem, 2em).
  • font-weight → grosor: normal, bold, o de 100 a 900.
  • font-style → italic para cursiva.
  • font-family → la fuente: "Arial", "Georgia", etc.
  • color → color del texto.
  • text-align → left, center, right, justify.
  • text-decoration → underline, line-through, none.
  • text-transform → uppercase, lowercase, capitalize.
  • letter-spacing → espacio entre letras.
  • line-height → altura de línea (interlineado).
h1 {
  font-family: 'Georgia', serif;
  font-size: 42px;
  font-weight: 800;
  color: #1572b6;
  text-align: center;
  letter-spacing: -1px;
  line-height: 1.1;
}

.subtitulo {
  font-style: italic;
  text-transform: uppercase;
  letter-spacing: 3px;
}

.tachado {
  text-decoration: line-through;
  color: gray;
}

Box Model: las cajas de CSS

El concepto que confunde a todos · 45 minutos

📦 En CSS todo es una caja

Cada elemento HTML es una "caja" con 4 capas concéntricas:

  • content → el contenido (texto, imagen).
  • padding → el ESPACIO INTERNO (entre el contenido y el borde).
  • border → el BORDE de la caja.
  • margin → el ESPACIO EXTERNO (entre la caja y otras).
.tarjeta {
  width: 300px;             /* ancho */
  height: 200px;            /* alto */

  padding: 20px;            /* interno: 20px en los 4 lados */
  border: 3px solid blue;  /* grosor estilo color */
  margin: 15px;             /* externo: 15px en los 4 lados */

  border-radius: 12px;      /* esquinas redondeadas */
  background: white;
}

/* También se puede especificar por lado: */
.otra {
  padding-top: 10px;
  padding-right: 20px;
  padding-bottom: 10px;
  padding-left: 20px;

  /* O todo en una línea: arriba derecha abajo izquierda */
  padding: 10px 20px 10px 20px;

  /* O incluso más corto: arriba/abajo izquierda/derecha */
  padding: 10px 20px;
}

Esta caja tiene padding: 20px (espacio interno) y border: 5px solid azul:

🟦 Soy el contenido. Mirá cómo el borde está separado de mí por el padding.
⚠️ Truco que cambia todo: agregá esto al inicio de tu CSS:
* { box-sizing: border-box; }
Esto hace que cuando ponés width: 300px, esos 300px incluyan el padding y border. Sin esto, el ancho real es width + padding + border, que es un dolor de cabeza.

📐 Bordes y esquinas

.caja-1 { border: 2px solid black; }
.caja-2 { border: 3px dashed red; }
.caja-3 { border: 4px dotted blue; }
.caja-4 { border: 5px double green; }

.redonda     { border-radius: 50%; }    /* círculo */
.suavizada   { border-radius: 12px; }   /* esquinas suaves */
.una-esquina { border-top-left-radius: 30px; }

Flexbox: alinear cosas en una línea

El sistema más usado para layouts · 1 hora

🎯 Para qué sirve Flexbox

Cuando necesitás poner cosas una al lado de la otra (o una arriba de la otra) y controlar cómo se alinean, Flexbox es la herramienta. Es el sistema más usado del CSS moderno.

El secreto es: ponés display: flex en el contenedor, y los hijos se ordenan automáticamente.

.contenedor {
  display: flex;                     /* ← la clave */
  gap: 10px;                        /* espacio entre hijos */

  justify-content: center;           /* eje horizontal */
  align-items: center;               /* eje vertical */
  flex-direction: row;               /* row | column */
}

📐 Los valores más útiles de justify-content

  • flex-start → todo al inicio (por defecto).
  • flex-end → todo al final.
  • center → todo al centro.
  • space-between → primero al inicio, último al final, los del medio repartidos.
  • space-around → con espacios iguales alrededor.
  • space-evenly → con espacios EXACTAMENTE iguales.

Esta combinación es ORO PURO. Cualquier elemento queda perfectamente centrado:

  display: flex;
  justify-content: center;
  align-items: center;
  height: 300px;
}

Grid: layouts en cuadrícula

El sistema para diseños complejos · 45 minutos

🎯 Cuándo usar Grid en lugar de Flexbox

Flexbox sirve para una dimensión (una línea o una columna). Grid sirve para dos dimensiones (filas Y columnas a la vez). Es perfecto para galerías, tableros y layouts de página. 

.galeria {
  display: grid;
  grid-template-columns: repeat(3, 1fr);  /* 3 columnas iguales */
  gap: 15px;                                  /* espacio entre celdas */
}

/* Otras formas de definir columnas: */
.layout-1 { grid-template-columns: 200px 1fr; }       /* fija + flexible */
.layout-2 { grid-template-columns: 1fr 2fr 1fr; }      /* proporcional */
.layout-3 { grid-template-columns: repeat(4, 1fr); }   /* 4 columnas */

Hover y transiciones

Efectos al pasar el mouse · 30 minutos

🖱️ El pseudo-selector :hover

Le agregás :hover al final de un selector para decir "estos estilos solo se aplican cuando el usuario PASA EL MOUSE por encima". 

.boton {
  background: blue;
  color: white;
  padding: 12px 24px;
  border-radius: 8px;

  /* ← LA CLAVE: hace que el cambio sea SUAVE */
  transition: all 0.3s ease;
}

.boton:hover {
  background: red;
  transform: scale(1.1);
}

⏱️ La propiedad transition

Define cómo se hace el cambio (suave, rápido, lento, etc.). Tiene 4 valores:

  • Qué propiedad cambia: all, color, transform...
  • Cuánto tarda: 0.3s, 1s, 500ms...
  • Curva: ease, linear, ease-in, ease-out.
  • Delay (opcional): cuánto espera antes de empezar.
/* Todos cambian con suavidad de 0.3 segundos */
.tarjeta {
  transition: all 0.3s ease;
}

/* Solo el color cambia, en 0.5s, lentamente al final */
.titulo {
  transition: color 0.5s ease-out;
}

/* Cambian dos cosas a la vez */
.imagen {
  transition: transform 0.4s, opacity 0.6s;
}

🎨 Ideas de efectos hover comunes

/* 1. Cambiar de color */
.btn:hover { background: red; }

/* 2. Agrandar */
.btn:hover { transform: scale(1.1); }

/* 3. Levantar */
.btn:hover { transform: translateY(-4px); }

/* 4. Rotar */
.btn:hover { transform: rotate(5deg); }

/* 5. Hacer transparente */
.btn:hover { opacity: 0.7; }

/* 6. Sombra que crece */
.btn:hover { box-shadow: 0 10px 20px rgba(0,0,0,0.2); }

Animaciones con @keyframes

Movimiento real y continuo · 45 minutos

🎬 ¿Diferencia entre transition y animation?

  • Transition → cambio entre dos estados (necesita un trigger como :hover).
  • Animation → secuencia que puede repetirse sola, sin trigger, con varios pasos.

🎯 Cómo se hace una animación

  1. Definir los pasos con @keyframes nombre { ... }.
  2. Aplicarla con animation: nombre tiempo modo; al elemento.
/* 1. Definimos la animación */
@keyframes flotar {
  0%   { transform: translateY(0); }
  50%  { transform: translateY(-15px); }
  100% { transform: translateY(0); }
}

/* 2. La aplicamos al elemento */
.cajita {
  animation: flotar 2s ease-in-out infinite;
  /*         nombre  duración  curva  cantidad */
}

Tres animaciones distintas corriendo en paralelo:

📋 Las 3 animaciones más útiles

/* 1. APARECER (fade-in) - el clásico */
@keyframes aparecer {
  from { opacity: 0; transform: translateY(20px); }
  to   { opacity: 1; transform: translateY(0); }
}

h1 {
  animation: aparecer 1s ease-out;
}


/* 2. GIRAR sin parar (loader) */
@keyframes girar {
  from { transform: rotate(0deg); }
  to   { transform: rotate(360deg); }
}

.loader {
  animation: girar 1s linear infinite;
}


/* 3. LATIR (corazón, notificación) */
@keyframes latir {
  0%, 100% { transform: scale(1); }
  50%      { transform: scale(1.2); }
}

.corazon {
  animation: latir 1s ease-in-out infinite;
}

📊 Valores de la propiedad animation

El orden es: nombre / duración / curva / delay / cantidad / dirección.

  • Cantidad: 1, 3, infinite...
  • Curva: linear, ease, ease-in, ease-out, ease-in-out.
  • Dirección: normal, reverse, alternate (va y vuelve).

Responsive: que se vea bien en celular

Media queries · 30 minutos

📱 ¿Qué es ser "responsive"?

Que tu sitio se adapte a distintos tamaños de pantalla: PC, tablet, celular. Hoy más de la mitad del tráfico web es desde celular, así que es obligatorio.

🎯 Las dos reglas de oro

1. Poné esta línea en el <head> de tu HTML (sin ella, nada va a funcionar bien en celular): 

<meta name="viewport"
      content="width=device-width, initial-scale=1.0">

2. Usá media queries en tu CSS para cambiar los estilos según el tamaño de pantalla. 

/* Estilos base (PC) */
.galeria {
  display: grid;
  grid-template-columns: repeat(4, 1fr);  /* 4 columnas */
  gap: 20px;
}

h1 {
  font-size: 48px;
}

/* === MEDIA QUERY: cuando la pantalla es chica (max 768px = tablet) === */
@media (max-width: 768px) {
  .galeria {
    grid-template-columns: repeat(2, 1fr);   /* 2 columnas */
  }
  h1 {
    font-size: 32px;
  }
}

/* === Para celular (max 480px) === */
@media (max-width: 480px) {
  .galeria {
    grid-template-columns: 1fr;   /* 1 sola columna */
  }
  h1 {
    font-size: 24px;
  }
}

📏 Los breakpoints más usados

  • max-width: 480px → celular.
  • max-width: 768px → tablet.
  • max-width: 1024px → laptop chica.
  • min-width: 1200px → pantalla grande.
💡 Tip de profesional: en el navegador, apretá F12 y después el ícono de celular/tablet (arriba a la izquierda). Te deja simular distintos dispositivos sin moverte de la silla.
